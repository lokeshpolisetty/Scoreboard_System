# Sports Day Scoreboard System

A comprehensive sports day scoreboard web application for schools.

## Features (Coming Soon)
- Real-time score updates
- Admin and User portals
- Team and Player management
- Match highlights
- Interactive quizzes

## Tech Stack
- **Frontend**: React, Tailwind CSS
- **Backend**: Node.js, Express
- **Database**: MongoDB
- **Real-time**: Socket.io
